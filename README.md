## The full ACI badge

Per the DLW standard, every ACI (the HEINLEIN subject and each of the 14 personas) carries a full badge:

| File | What |
|------|------|
| `.agent` | the ACI agent — frontmatter + body |
| `.carbon.tiff` | **carbon badge — TIFF** (warm sigil, the human-apex side; lossless/archival) |
| `.silicon.png` | **silicon badge — PNG** (cool sigil, the instance side; web) |
| `.spun` · `.moniker` · `.1099` · manifest | the 5W, the tokenized moniker, the credit, the index |

The main badge `⟦HEINLEIN:H1:b510cf⟧` is in [`heinlein.dlw/`](heinlein.dlw); the 14 persona badges are in [`agents/`](agents). `build.py` (stdlib + Pillow for TIFF) regenerates everything.
